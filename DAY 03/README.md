# Day 03 Summary

Today I learned how to use CSS Flexbox to create responsive and well-structured layouts.

### What I Learned
- Used `display: flex` to create flexible layouts.
- Applied `justify-content: space-between` to position the logo and navigation links.
- Used `align-items: center` for proper vertical alignment.
- Created a horizontal navigation menu using Flexbox.
- Used the `gap` property to maintain equal spacing between navigation links.
- Centered the hero section both horizontally and vertically using `justify-content` and `align-items`.
- Explored `flex-direction: column` to stack elements vertically.
- Added a simple hover effect to navigation links for better user interaction.

### Key Takeaways
This challenge helped me understand how Flexbox simplifies alignment and spacing in web layouts. I also learned how to build a clean navigation bar and a centered hero section, which are essential components of modern websites.
