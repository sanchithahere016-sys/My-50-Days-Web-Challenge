# Day 7 - Form UI & Interactive States

## What I Learned
- Created an HTML membership application form.
- Styled input fields, textarea, and select elements.
- Used CSS pseudo-classes like :focus, :hover, :disabled, :valid, and :invalid.
- Improved accessibility using labels linked with inputs.
- Designed a responsive and user-friendly form interface.

## Technologies Used
- HTML5
- CSS3

## Outcome
Built a clean, responsive membership application form with interactive UI states that provide visual feedback during user interaction.
