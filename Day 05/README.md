# Day 05 - CSS Grid Architecture

## Overview

In this challenge, I learned how to build a responsive layout using CSS Grid. I created an Initiatives Gallery consisting of project cards arranged in equal columns with proper spacing and styling.

## What I Learned

- How to use `display: grid`
- Creating equal columns with `grid-template-columns`
- Using `repeat()` and `1fr`
- Adding spacing with `gap`
- Styling cards using padding, border-radius, and box-shadow
- Creating a featured card using `grid-column: span 2`

## Technologies Used

- HTML5
- CSS3
