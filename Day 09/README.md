# Day 09 – Accessibility (a11y) Standards

## Objective
Improve the accessibility of the Community Platform website by following WCAG principles.

## Features
- Skip to Main Content link
- Semantic HTML
- Accessible image alt text
- Keyboard navigation with :focus-visible
- Better color contrast

## Technologies
- HTML5
- CSS3

## Outcome
Built a more accessible and inclusive website that supports keyboard users and screen readers.
